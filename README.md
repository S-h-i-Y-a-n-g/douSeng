在本次项目实践中，我们通过项目进一步学习了知识点，并且使用各位老师们教授的内容，从语言基础到拓展技术，实践完成此次项目，收获颇多，感谢青训营的这次机会

该项目基于gin-admin框架进行开发。采用三层架构将 API、SERVICE、MODEL、COMMON、STATIC 等层次分开，通过各种方法进行连接调度，极大地提高了团队间的效率。

使用技术：GIN,GORM,JWT,REDIS,CASBIN,DOCKER,OSS

然而由于时间较为紧张，很多设想中的优化并未进行，例如视频上传分块未完善，自动设置头像背景，抽取通用类等等，后续将会慢慢完成

项目最终实现了：基础接口、扩展接口Ⅰ和扩展接口Ⅱ。性能测试尚可。
